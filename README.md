# Douyin Clone (TikTok) - Premium Web Edition

A high-performance TikTok/Douyin clone built with React, MobX, and React Router 6.

## Technology Stack
- **Framework**: React
- **State Management**: MobX
- **Routing**: React Router 6
- **Security**: HSTS (via Netlify/Tengine) & CryptoJS
- **PWA**: Fully offline capable & installable
- **Highlighting**: Prism.js
- **Assets**: Tanx DAM Integration (Mock)

## Features
- Vertical Video Feed with native Scroll Snap.
- Premium Glassmorphism UI.
- Secure metadata handling.
- Optimized for Netlify deployment.
